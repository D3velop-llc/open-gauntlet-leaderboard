# OpenGauntlet — leaderboard site

Which local model feels the most human? Rendered leaderboard for
[opengauntlet.com](https://opengauntlet.com) — conversational-EQ rankings of
locally-hostable LLMs: Bradley-Terry Elo with bootstrap confidence intervals,
head-to-head win rates, and a cross-provider judge-calibration panel.

This repo holds the generated static site only; the benchmark harness that
produces it lives in a private repository.
